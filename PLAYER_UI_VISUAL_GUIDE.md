## Color Scheme

### Title Screen
- **Title Text:** Cyan/Blue (0.3, 0.8, 1.0) with pulsing effect
- **Subtitle:** Light Gray (0.7, 0.7, 0.8)
- **Background:** Black with 85% opacity overlay
- **Stars:** White/Cyan with glow effect

### Main Game
- **Player Status Panel:** 80% opacity background
- **Text Colors:**
  - Headers: Cyan (0.3, 0.8, 1.0)
  - Normal text: White
  - Important values: Yellow/Gold
  - Warnings: Orange/Red
  
### HUD Elements
- **Good status:** Green
- **Warning status:** Yellow
- **Critical status:** Red
- **UI accents:** Cyan/Blue

### Futuristic HUD
- **Frames:** Cyan with transparency
- **Radar:** Green blips for friendlies, Red for enemies
- **Gauges:** Gradient from green (full) to red (empty)

## Key Visual Features

1. **Starfield Background:** Procedural stars with parallax effect
2. **Voxel Ships:** Colored blocks representing ship components
3. **Holographic UI:** Semi-transparent panels with glowing edges
4. **Animated Elements:** 
   - Title screen pulsing
   - Rotating decorative stars
   - Blinking text
   - Radar sweep animation
   - Progress bar fills
5. **Debug Overlays:** FPS counter, entity count, performance metrics

## Control Mode Indicators

**Camera Mode (Default):**
- HUD shows "WASD - Move Camera"
- Mouse controls camera rotation

**Ship Control Mode (Press C):**
- HUD shows "WASD - Thrust"
- Console message: "Control Mode: Ship Control"
- Camera follows ship from behind

## Responsive Design

All UI panels:
- Auto-resize based on content
- Can be dragged (when not locked)
- Semi-transparent backgrounds
- Consistent styling
- ImGui immediate-mode updates
